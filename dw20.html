General access

To save on cost, the server is set to shut down when idle. It will boot up when a request is made to the server. There are a few ways to do this, any one of these will work:

  1. Add the server to your server list. Open the server list. It will time out until it boots.
  2. Try to visit dw20.mcserver.cyou. It won't find a server.
  3. Visit this page.
Then wait a few minutes and the server should be booted.

Download the modpack

  1. Download the FTB App https://feed-the-beast.com/ftb-app.
  2. Search for and download Direwolf20 1.19 through the FTB App.

  1. Download MultiMC https://multimc.org/.
